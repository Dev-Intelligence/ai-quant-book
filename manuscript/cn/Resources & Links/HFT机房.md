# HFT 机房与共址服务

> 高频交易（HFT）需要将服务器部署在交易所附近的机房，以获得最低延迟。本文介绍主要交易所的共址服务和通信服务商。

---

## 主要服务商

| **服务商**                                | **服务内容**      | **特色**              |
| -------------------------------------- | ------------- | ------------------- |
| **Equinix**                            | 共址、裸机租赁、专线    | 覆盖最全、HFT标准          |
| **Hudson Fiber Network**               | 专线光纤          | 美东交易所低延迟线路          |
| **McKay Brothers**                     | 微波通信          | 极限低延迟，跨洲线路（如伦敦-芝加哥） |
| **Colt**                               | 亚太区光纤高速连接     | 东京、香港、新加坡机房         |
| **专用低延迟光纤服务商** | 专用光纤（如芝加哥到纽约） | 延迟 ~13ms（如 Jump Trading 旗下线路） |
